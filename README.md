# Environment Setting

### Install
Install conda

```
conda create -n "ml2" python=3.9
conda activate "ml2"
conda install pytorch==1.12.1 torchvision==0.13.1 torchaudio==0.12.1 cudatoolkit=11.3 -c pytorch
```

Set VSC default python interpreter as "ml2" python 3.9.10
